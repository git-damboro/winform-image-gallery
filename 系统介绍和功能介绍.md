# WinForms Image Gallery 系统介绍和功能介绍

## 一、系统介绍

WinForms Image Gallery 是一个 Windows 本地图片缩略图浏览系统，主要面向大量图片的快速浏览、筛选、预览和基础管理场景。系统使用 `.NET 8.0` 和 WinForms 开发，可以在 Windows 10 / Windows 11 环境运行。

系统不会把图片复制到程序目录，而是读取用户选择的本地图片路径，并在下次启动时自动恢复上次导入的图片列表。这样可以减少磁盘占用，也便于直接管理用户本地已有的图片文件。

## 二、适用场景

| 场景 | 说明 |
|---|---|
| 本地图片浏览 | 快速查看本机文件夹中的大量图片 |
| 图片素材筛选 | 按图片类型筛选 PNG、JPG、BMP、GIF、WEBP 等文件 |
| 缩略图预览 | 通过缩略图网格快速浏览图片内容 |
| 大图查看 | 双击图片后进入大图窗口查看细节 |
| 批量选择 | 支持全选当前可见图片并删除选中项 |

## 三、核心功能

| 功能模块 | 功能说明 |
|---|---|
| 图片导入 | 支持一次选择多张图片导入 |
| 缩略图显示 | 以网格方式展示图片缩略图 |
| 滚动浏览 | 支持鼠标滚轮和滚动条实时浏览 |
| 类型筛选 | 支持多选图片类型，只影响当前画布显示 |
| 信息显示控制 | 可选择显示图片名称、大小、类型、尺寸，也可以只显示图片 |
| 缩略图风格 | 支持 default、rounded、shadow、border、polaroid、glass、crystal、neon、minimal |
| 选中状态 | 选中图片时显示更明显的深蓝覆盖和直角边框 |
| 大图预览 | 支持双击打开大图，鼠标点击或键盘左右键切换 |
| 会话保存 | 自动保存图片路径列表和当前缩略图风格 |
| 任务进度 | 导入和启动恢复时显示任务状态、当前进度和当前文件 |

## 四、缩略图风格介绍

| 风格 | 视觉效果 |
|---|---|
| default | 保持默认卡片样式 |
| rounded | 更明显的圆角卡片 |
| shadow | 更强的阴影和层次 |
| border | 清晰的描边边框 |
| polaroid | 拍立得照片样式 |
| glass | 柔和半透明玻璃质感 |
| crystal | 冷色渐变边框、高光和轻微发光的水晶质感 |
| neon | 蓝紫色霓虹发光效果 |
| minimal | 极简扁平风格 |

## 五、新电脑运行说明

在新电脑运行时，需要先安装 `.NET 8 Desktop Runtime`。安装后解压发布包，双击 `ImageGallery.App.exe` 即可启动。

如果新电脑需要继续开发源码，则需要安装 `.NET SDK 8.0` 和 Visual Studio 2022 的“.NET 桌面开发”工作负载。

## 六、注意事项

- 程序保存的是图片路径，不会复制图片文件。
- 如果图片文件移动到其他位置，需要重新导入。
- 大量图片首次加载会读取图片尺寸和格式，加载期间会显示进度。
- 建议把大量图片放在本地磁盘，避免网络盘读取过慢。
- 当前发布包为框架依赖发布，新电脑需要安装 `.NET 8 Desktop Runtime`。
